## VALUE 0S - Mission / Vision / Code

**Mission**
- 他者に支配されない「自律的な価値判断」を生む装置をつくる。

**Vision**
- 誰もが自分で価値を決め、笑いと驚きを通じて生きる世界。

**Code (行動原則)**
- 創造=衝動エネルギーを正に変換し、迷える個人に"判断の武器"を渡す。

# ChatGPTとの対話

- 1.あなたにとって「おもしろい」とは何ですか？
    - ダウンタウンの松本人志が確立をしたお笑いの価値観。
- 2.なぜアウトプットが恥に結びつきますか？
    - 逃げたくなるから。自分に自信がないから。
- 3.人があなたに反応する時、何が怖いですか？
    - 陰口や裏切られることが怖い。
- 4.それでも創りたいのはなぜですか？
    - 自分のやりたいことであり、衝動として考えてしまうから。
- 5.あなたが最も「救われる」と感じる瞬間は？
    - 自分の価値観を確立してそこで生きることができるとき。
- 6.逆に最も「破滅しそう」と感じる瞬間は？
    - 衝動性や暴力性が抑えられなくなり、表に表出するとき。
- 7.あなたの人生で一貫しているテーマは？
    - 相手がサプライズとして驚きの反応をしたときはうれしく感じる。
    - それが前までは、お笑いで相手を笑わせることだった。
- 8.あなたが作りたい世界はどんな世界ですか？
    - おもしろい-つまらないという価値判断に支配されることなく、
    - 極論を言えば、自律的にみんなが笑い自分で価値を決める世界。
- 9.逆に絶対に許せない世界は？
    - 全体主義(ファシズム)として頂点に君臨をして支配している世界。
- 10.誰か1人にこのプロダクトを届けるなら、どんな人？
    - 今のお笑いに対して懐疑しているが、なぜなのかがわからない人。


## 抽出された10の"思想OSコア原則"
- 1.価値は自分で決める（Value Autonomy）
- 2.反応は恐怖でなく対話である（Fear Reframing）
- 3.創造と暴力は同じエネルギー源（Energy Unity）
- 4.驚きは存在の肯定（Existence via Surprise）
- 5.恥は裏切りの記憶の再演（Trauma Echo）
- 6.笑いは支配から自律するための武器（Comedy as Liberation）
- 7.世界は極端から価値を作る（Edge Thinking）
- 8.集団支配には抗う（Anti-Fascism）
- 9.価値判断を再教育する（Relearn Value System）
- 10.迷える個人に道具を渡す（Empower the Lost）
