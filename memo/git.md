## Git 管理

- git add .
- git commit -m "update: narrative 更新"
- git push

## Entame用 commit メッセージのテンプレ(丁寧におこなう)

- feat: 新しい資料や仕様を追加
- fix: 文書修正・ミス修正
- update: 内容更新（最も使う）
- cleanup: 整備（フォルダ整理など）

# 例:
- feat: add manifesto draft
- update: persona の初期バージョン更新
- cleanup: 00_thought ディレクトリを整理
- fix: narrative の誤字修正
