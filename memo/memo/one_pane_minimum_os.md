# A4 One-Pane Minimum OS

---

## 1. Mission / 天命（1行）

「衝動・羞恥・恐怖を素材に、世界の感受性を変える芸術＝Entameを創造する」

---

## 2. Core Insight / 核となる気づき

- 衝動性の中心には「胸の谷間の線」がある（恥・恐怖・興味の交点）
- 自慰 = 瞑想 = 呼吸 = 創造衝動 の同一性
- ニーチェ / バタイユ / D+G / ユング の思想が全て「禁忌 → 創造」へ向かう
- 恐怖・恥を越境した先に“作品”がある
- これら全てを **一枚の創造OS** に集約することが、あなたの道

---

## 3. Value OS（価値の根）

- 禁忌の肯定：恥・恐怖・影（Shadow）は創造の源泉
- 越境：境界を越えたときエロースが成立する
- 変容：性・志向・衝動の転換そのものが作品
- 誠実さ：もっとも隠してきたものが最も価値になる
- 単位化：minimum（最小行動）で世界を動かす

---

## 4. Skill Root（毎日の最低単位）

- Design：Figmaで20分、1 UI（Entameの断片）
- Coding：1コミット（TS / Rust / Python どれでも可）
- Thought：モーニングページで“影”を言語化
- Business：リーン1インプット
- Output：どこでもいいから1行アウトプット

---

## 5. Entame Root（思想 → プロダクト 変換OS）

1. 思想（00_thought）

    - value_os / manifesto / narrative / trauma_fear_notes
2. 設計（01_design

   - persona / user_journey / architecture
3. プロトタイプ（02_prototype

   - Figma → Next.js → Vercel
4. 出力（03_output_log）

   - Output Prototype / daily_output

🔗 この4層が1本の管（One Pane）で繋がることで “世界観 = サービス” になる。

---

## 6. Daily Loop / 毎日のOS

```
[1] 呼吸（自慰=瞑想）で影を観察
[2] minimum output を1つ出す
[3] Entame に1コミット
[4] 日報に1行だけ記録
```

"大きくやろうとしない。minimum だけを積む"

---

## 7. KPI（定量ではなく、感覚KPI）

- ● と ○ が切り替わる瞬間があったか
- 衝動性を“対象化”できた瞬間があったか
- 恥を避けず、素材として扱えたか
- 作品に1mmでも近づいた感覚があったか

---

## 8. Entame Vision（2030の姿）

「写真や絵の人物に“語らせる”世界最小のエンタメプラットフォーム」

- Skeb × ボカロ × ミーム × AI
- "影" と "衝動" の世界観が UI に宿る
- あなた自身の衝動の物語が、世界の創作者のOSになる

---

## 9. Anti-Failure（破滅を避けるための原則）

- 逃避（スマホ・遅寝・無行動）はすべて“恥の回避”
- 衝動は抑圧すると暴走する、**作品に変換することで中和**される
- 「行動しない＝最大の破滅」
- 最小行動こそ最大の安全装置

---

## 10. 最終行（あなたの核心）

「衝動の中心に触れたとき、作品は生まれる。」
