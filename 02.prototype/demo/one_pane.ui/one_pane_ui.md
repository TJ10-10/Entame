## UI名: "Speak from the Shadow"
    - ワンペイン構造 (1画面/1操作/1フィードバック)

---

# 1.画像アップロード (中心)
    - ユーザーが 1枚の画像（人物、イラスト、胸の谷間の線、象徴的なもの、
    - 何でも）をアップする。
    - あなたの概念の「中心（象徴）」を入力する場所。

# 2.キャラの視点を選ぶ (右側 or 下）
    - プルダウン形式（Shadow / Eros / Revolt / Father / Child / Void など）
    - ここで「どの人格が語るか」が決まる。

# 3.セリフ生成ボタン（“Talk”）
    - 画像＋視点 → セリフを生成する。
    - これが「恥・衝動性・影・エロース」を 言語化したアウトプット になる。

# 4.結果表示エリア
    - 人物の横に吹き出しでテキストを表示。
    - たった1行でもOK。
    - “影が語る” をミニマムに体験させる。
